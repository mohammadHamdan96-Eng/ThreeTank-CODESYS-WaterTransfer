## CV Project Entry - Stronger Engineering Version

**Three-Tank Water Transfer System - CODESYS PLC/HMI V1 Simulation**  
Designed and implemented a PLC/HMI process automation simulation in CODESYS for a three-tank water-transfer system. Developed the main control sequence in Ladder Diagram with step logic (S0-S70), valve and pump permissives, simulated feedback signals, fault state handling, reset logic, tank level alarms, and a basic manual mode. Used Structured Text to simulate analog tank levels and CODESYS Visualization to create an HMI overview for monitoring and operation.
